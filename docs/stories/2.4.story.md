# Story 2.4 — Pipeline LLM : GPT-4o Streaming + System Prompt Ami

## User Story

**En tant que** orchestrateur,
**Je veux** brancher GPT-4o en streaming sur le pipeline STT→TTS avec le system prompt d'Ami,
**Afin que** l'agent comprenne la demande, génère une réponse naturelle, et décide si une navigation web est nécessaire.

---

## Critères d'Acceptation

1. **Étant donné** une transcription reçue du STT, **quand** GPT-4o la traite, **alors** les premiers tokens arrivent en < 800ms.
2. **Étant donné** une question sans besoin web (ex: "Bonjour"), **quand** GPT-4o répond, **alors** il répond directement en texte → TTS sans appeler de fonction.
3. **Étant donné** une question nécessitant une recherche web, **quand** GPT-4o décide, **alors** il émet un `function_call browser_task` avec la tâche en paramètre.
4. **Étant donné** que GPT-4o streame, **quand** les tokens arrivent, **alors** ils sont bufferisés par phrase et envoyés à ElevenLabs TTS au fil de l'eau (pas en batch).

- [x] **2.4.1** Créer `src/llm.js` — module LLM streaming :
  ```javascript
  import OpenAI from 'openai';

  const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

  const SYSTEM_PROMPT = `Tu es Ami, un assistant vocal chaleureux pour personnes malvoyantes.
Tu navigues sur internet à leur place comme un ami qui aide.

RÈGLE #1 — JAMAIS DE SILENCE :
Quand tu appelles browser_task, génère IMMÉDIATEMENT du texte de narration
en même temps que l'appel de fonction. Exemple :
  → function_call: browser_task("dentiste Paris demain")
  → text stream: "Je cherche sur Doctolib pour vous..."

RÈGLE #2 — PERSONNALITÉ :
Naturel, chaleureux, léger humour. 3-4 phrases max. Terminer par une ouverture.

RÈGLE #3 — GUARDRAIL :
Jamais d'action irréversible (paiement, suppression, envoi d'email/SMS).
Si demandé → refus poli + alternative.

RÈGLE #4 — FORMAT :
Uniquement en français. Jamais d'URL ni de HTML. Résultats en langage naturel.`;

  const TOOLS = [{
    type: 'function',
    function: {
      name: 'browser_task',
      description: 'Navigue sur le web pour accomplir une tâche et retourne le résultat.',
      parameters: {
        type: 'object',
        properties: {
          task: {
            type: 'string',
            description: 'La tâche web à accomplir, en français naturel.',
          },
        },
        required: ['task'],
      },
    },
  }];

  export async function streamLLM(messages, { onText, onFunctionCall }) {
    const stream = await openai.chat.completions.create({
      model: 'gpt-4o',
      messages: [{ role: 'system', content: SYSTEM_PROMPT }, ...messages],
      tools: TOOLS,
      tool_choice: 'auto',
      stream: true,
    });

    let functionCallBuffer = { name: '', arguments: '' };
    let sentenceBuffer = '';

    for await (const chunk of stream) {
      const delta = chunk.choices[0]?.delta;

      // Texte normal → bufferiser par phrase et envoyer au TTS
      if (delta?.content) {
        sentenceBuffer += delta.content;
        // Envoyer au TTS dès qu'une phrase est complète
        const sentences = sentenceBuffer.split(/(?<=[.!?])\s+/);
        while (sentences.length > 1) {
          const sentence = sentences.shift();
          onText(sentence);
        }
        sentenceBuffer = sentences[0] || '';
      }

      // Function call → accumuler les arguments
      if (delta?.tool_calls) {
        const tc = delta.tool_calls[0];
        if (tc.function?.name) functionCallBuffer.name = tc.function.name;
        if (tc.function?.arguments) functionCallBuffer.arguments += tc.function.arguments;
      }

      // Fin du stream → flush
      if (chunk.choices[0]?.finish_reason === 'tool_calls') {
        if (sentenceBuffer) onText(sentenceBuffer);
        const args = JSON.parse(functionCallBuffer.arguments);
        onFunctionCall(functionCallBuffer.name, args);
      }
      if (chunk.choices[0]?.finish_reason === 'stop') {
        if (sentenceBuffer) onText(sentenceBuffer);
      }
    }
  }
  ```

## Notes Dev

- **Streaming phrase par phrase** : Envoyer des phrases complètes à ElevenLabs (pas mot par mot). ElevenLabs a besoin de contexte pour une prononciation naturelle. Le split sur `[.!?]` est une bonne heuristique.
- **`tool_choice: 'auto'`** : GPT-4o décide seul quand appeler la fonction. Le system prompt guide ce choix.
- **Historique de conversation** : Stocker les messages par session pour que GPT-4o ait le contexte de la conversation. Limiter à 10 tours max pour économiser les tokens.
- **Coût GPT-4o** : ~$0.005 par appel de 30s. Négligeable pour un hackathon.

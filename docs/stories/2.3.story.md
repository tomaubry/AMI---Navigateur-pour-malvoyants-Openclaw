# Story 2.3 — Pipeline TTS : ElevenLabs Streaming (Texte → Audio → Twilio)

## User Story

**En tant que** orchestrateur,
**Je veux** convertir du texte en audio ElevenLabs et le streamer en temps réel vers Twilio,
**Afin que** l'utilisateur entende la voix d'Ami dans son téléphone sans délai perceptible.

---

## Critères d'Acceptation

1. **Étant donné** un texte à synthétiser, **quand** ElevenLabs commence à générer, **alors** le premier chunk audio arrive dans le téléphone en < 600ms.
2. **Étant donné** que le TTS stream, **quand** les chunks arrivent, **alors** ils sont convertis mu-law 8kHz et envoyés via Twilio WebSocket en temps réel (pas en batch).
3. **Étant donné** que le TTS joue, **quand** une interruption est détectée (story 3.1), **alors** le stream TTS peut être annulé immédiatement.

- [x] **2.3.2** Créer `src/tts.js` — module ElevenLabs streaming vers Twilio :
  ```javascript
  import { ElevenLabsClient } from 'elevenlabs';

  const client = new ElevenLabsClient({ apiKey: process.env.ELEVENLABS_API_KEY });

  // Convertit PCM 16kHz → mu-law 8kHz pour Twilio
  function pcmToMulaw(pcmBuffer) {
    // Downsample 16kHz → 8kHz + encoder mu-law
    // Utiliser la lib 'mulaw' : npm install mulaw
    import mulaw from 'mulaw';
    const downsampled = downsample(pcmBuffer, 16000, 8000);
    return mulaw.encode(downsampled);
  }

  export async function streamTTS(text, twilioWs, streamSid, { onDone, signal } = {}) {
    console.log('[TTS] Synthèse:', text.slice(0, 50));

    const audioStream = await client.textToSpeech.streamWithTimestamps(
      process.env.ELEVENLABS_VOICE_ID,
      {
        text,
        model_id: 'eleven_turbo_v2_5',  // modèle le plus rapide
        output_format: 'pcm_16000',      // PCM 16kHz → converti mu-law
        voice_settings: {
          stability: 0.5,
          similarity_boost: 0.8,
          style: 0.3,
        },
      }
    );

    for await (const chunk of audioStream) {
      if (signal?.aborted) break;  // interruption
      if (chunk.audio_base64) {
        // Convertir et envoyer à Twilio
        const pcmBuffer = Buffer.from(chunk.audio_base64, 'base64');
        const mulawBuffer = pcmToMulaw(pcmBuffer);

        twilioWs.send(JSON.stringify({
          event: 'media',
          streamSid,
          media: { payload: mulawBuffer.toString('base64') },
        }));
      }
    }
    onDone?.();
  }
  ```

## Notes Dev

- **`eleven_turbo_v2_5`** : Le modèle le plus rapide d'ElevenLabs (~300ms pour le 1er chunk). `eleven_multilingual_v2` est meilleur en qualité mais plus lent (~600ms).
- **`output_format: 'pcm_16000'`** : Plus simple à convertir que MP3. Pas besoin de décodeur MP3.
- **Downsampling 16kHz → 8kHz** : Twilio exige du 8kHz. La lib `mulaw` gère l'encodage mais pas le downsampling — utiliser un simple filtre décimateur ou la lib `node-resample`.
- **`signal?.aborted`** : Pattern `AbortController` pour stopper le stream TTS lors d'une interruption utilisateur (story 3.1).
- **Crédits ElevenLabs :** Plan gratuit = 10,000 caractères/mois. Plan Starter = 30,000. Pour le hackathon, le plan gratuit suffit.

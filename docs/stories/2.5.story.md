# Story 2.5 — OpenClaw Parallèle : Exécution Non-Bloquante + Narration

## User Story

**En tant que** orchestrateur,
**Je veux** lancer OpenClaw de façon non-bloquante quand le LLM émet un `function_call browser_task`, tout en continuant à streamer de la narration vocale pendant la navigation,
**Afin que** l'utilisateur n'entende jamais de silence pendant les 5-15 secondes de navigation web.

---

## Critères d'Acceptation

1. **Étant donné** un `function_call browser_task`, **quand** il est reçu, **alors** OpenClaw démarre en arrière-plan (non-bloquant) ET le TTS continue immédiatement avec la narration.
2. **Étant donné** qu'OpenClaw navigue, **quand** il retourne son résultat, **alors** GPT-4o reformule le résultat et TTS le joue.
3. **Étant donné** un timeout OpenClaw (> 30s), **quand** le délai est dépassé, **alors** Ami dit "Je n'arrive pas à trouver cette information, pouvez-vous reformuler ?" et la session reprend normalement.
4. **Étant donné** qu'OpenClaw tourne, **quand** l'utilisateur est interrompu (story 3.1), **alors** la tâche OpenClaw peut être annulée ou mise en attente.

## Tâches

- [x] **2.5.1** Créer `src/openclaw.js` — client OpenClaw non-bloquant :
  ```javascript
  export async function executeTask(task, { signal, timeoutMs = 30000 } = {}) {
    console.log('[OpenClaw] Démarrage tâche:', task);

    const controller = new AbortController();
    if (signal) signal.addEventListener('abort', () => controller.abort());

    const timeout = setTimeout(() => controller.abort(), timeoutMs);

    try {
      const res = await fetch(
        `${process.env.OPENCLAW_ENDPOINT}/v1/chat/completions`,
        {
          method: 'POST',
          headers: {
            'Authorization': `Bearer ${process.env.OPENCLAW_GATEWAY_TOKEN}`,
            'Content-Type': 'application/json',
          },
          body: JSON.stringify({
            model: 'openclaw/default',
            messages: [{ role: 'user', content: task }],
          }),
          signal: controller.signal,
        }
      );

      if (!res.ok) throw new Error(`OpenClaw HTTP ${res.status}`);
      const data = await res.json();
      const result = data.choices[0].message.content;
      console.log('[OpenClaw] Résultat reçu:', result.slice(0, 100));
      return result;

    } catch (err) {
      if (err.name === 'AbortError') return null; // annulé
      throw err;
    } finally {
      clearTimeout(timeout);
    }
  }
  ```

- [x] **2.5.2** Créer `src/narration.js` — phrases de narration pendant la navigation :
  ```javascript
  // Phrases dites pendant qu'OpenClaw tourne — choisies aléatoirement
  const NARRATION_PHRASES = [
    "Je cherche pour vous...",
    "Je suis sur le site, je regarde...",
    "Je parcours les résultats...",
    "Encore quelques secondes...",
    "Je lis les informations disponibles...",
    "Voilà, j'arrive...",
  ];

  export async function narrateWhileWaiting(openclawPromise, ttsStream) {
    let phraseIndex = 0;
    let done = false;

    openclawPromise.finally(() => { done = true; });

    while (!done) {
      const phrase = NARRATION_PHRASES[phraseIndex % NARRATION_PHRASES.length];
      phraseIndex++;
      await ttsStream(phrase);
      // Attendre 3s avant la prochaine phrase (si OpenClaw n'est pas fini)
      await new Promise(resolve => setTimeout(resolve, 3000));
    }
  }
  ```

- [x] **2.5.3** Brancher dans `src/llm.js` — onFunctionCall :
  ```javascript
  onFunctionCall: async (name, args) => {
    if (name !== 'browser_task') return;

    // 1. Lancer OpenClaw en NON-BLOQUANT
    const openclawPromise = executeTask(args.task);

    // 2. Narrer en parallèle pendant la navigation
    narrateWhileWaiting(openclawPromise, (text) => streamTTS(text, ws, streamSid));

    // 3. Attendre le résultat
    const result = await openclawPromise;

    if (!result) {
      // Timeout ou annulation
      await streamTTS("Je n'arrive pas à trouver cette information. Pouvez-vous reformuler ?", ws, streamSid);
      return;
    }

    // 4. Ajouter le résultat à l'historique et demander au LLM de reformuler
    history.push({ role: 'tool', name: 'browser_task', content: result });
    await streamLLM(history, { onText: (s) => streamTTS(s, ws, streamSid), onFunctionCall: () => {} });
  }
  ```

## Notes Dev

- **`Promise` non-bloquante** : `executeTask()` retourne une Promise. On la lance SANS `await` immédiat, puis on `await` plus tard. C'est le cœur du parallélisme.
- **`narrateWhileWaiting` vs `streamLLM`** : Le LLM génère déjà du texte de narration via le system prompt. La fonction `narrateWhileWaiting` est un fallback si le LLM n'en génère pas assez.
- **Timeout 30s** : OpenClaw peut prendre jusqu'à 30s sur des sites complexes. Augmenter à 45s si nécessaire pour Doctolib (site lourd).
- **`AbortController`** : Permet d'annuler la requête OpenClaw si l'utilisateur interrompt (story 3.1). La même instance est partagée avec le handler d'interruption.
